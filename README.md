### 操作說明
- 單人/加速模式  
←→：左右移動  
↓：加速下降  
↑：順時針旋轉  
/：逆時針旋轉  
c：Hold  
Space：硬掉落  
P：暫停  
- 雙人模式  
AD | ←→：左右移動  
S | ↓：加速下降  
W | ↑：順時針旋轉  
E | /：逆時針旋轉  
C | ,：Hold  
X | .：硬掉落
